# Hand Gesture Volume Control

This project controls system volume using hand gestures through a webcam.
It uses MediaPipe for hand tracking, OpenCV for video processing,
PyCaw for volume control, and Matplotlib for real-time visualization.

## Features
- Gesture-based volume control
- Two-hand interaction
- Lock and unlock mechanism
- Real-time volume graph
- FPS monitoring

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyCaw
- Matplotlib
